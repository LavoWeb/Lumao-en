Développement Akeneo

Akeneo est un Product Information Management. Avec cet outil, vous pouvez gérer des collections de produits très facilement. Pour tous les formats, que ce soit web, print, les données ERP… Le principe est que le PIM contient toutes les informations des produits et est le maitre de l’information. Il est souvent couplé avec un Data Asset Management qui stocke les médias (photos, vidéos, musiques…).
